sparrows.py

Cиситемные требования: Python3.*

Необходимые модули: threading, queue, time

Настройка: Изменение параметров в начале кода
			парраметр Time отвечает за скорость работы программы
				(Time = 60 соответствует устловию задания)
			Прочие параметры соответсвуют условию

Классы:
	Grandmother 
		Поток при инициалиации получающий параметры:
			period - соответствует парраметру Q из условия
			dataQueue - очередь из хлебных крошек
			N - соответсвует параметру N из условия
			Time - описан в настройках
		
		Реализует бабушку из условия
		
	Sparrow
		Поток при инициализации получающий параметры: 
			P - соответствует парраметру P из условия
			mutex - объект блокировки
			Id - номер потока(воробья)
			dataQueue - очередь из хлебных крошек
			now_eats - номер воробья отлетевшего с крошкой 

		Реализует воробья из условия
	
	Bench
		Поток при инициализации получающий параметры:
		M - соответствует парраметру M из условия
		P - соответствует парраметру P из условия
		N - соответствует парраметру N из условия
		Q - соответствует парраметру Q из условия 
			(если не задан, то равен N // P ) 
		Time - сописан в настройках (если не задан, то равен 60)
		
		Реализует скамейку из условия.
		Инициализирует и запускает Grandmother и Sparrow 
			(в том числе прилет новых)
		
Запуск осушествляется прямым запуском sparrows.py
